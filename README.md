## How to use
1. Clone the repository:
   ```bash
   git clone https://github.com/Nikipantdtu/Integrated-Energy-Grids-Project_Group-30.git
   cd Integrated-Energy-Grids-Project_Group-30
   ```
2. Open the notebooks with Jupyter:
   ```bash
   jupyter lab
   ```
   (or `jupyter notebook`)
